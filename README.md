# El-Claro-de-la-Selva
¡Hora de conocerse, madreselvers! Aquí encontraréis preguntas para conectar con aquellas personas con las que compartiréis mesa hoy... ¿Quién empieza? Miraos a los ojos y la primera persona que parpadee deberá responder la primera pregunta. Después, continuaréis en el sentido de las agujas del reloj. ¡Disfruten... y buen provecho!
